📝 क्लाउड ऑटो सिंक मार्गदर्शक (How to Setup Cloud Backup)

पायरी १: Google Apps Script उघडा आणि नवीन प्रोजेक्ट तयार करा.
पायरी २: खालील कोड कॉपी करून तेथे पेस्ट करा: 
function doPost(e) {
  var p = JSON.parse(e.postData.contents);
  var props = PropertiesService.getScriptProperties();
  if (p.action === 'save') {
    props.setProperty('vishwa_hardware_data', JSON.stringify(p.state));
    return ContentService.createTextOutput("Saved Successfully");
  }
}
function doGet(e) {
  var props = PropertiesService.getScriptProperties();
  var d = props.getProperty('vishwa_hardware_data') || '{}';
  return ContentService.createTextOutput(d).setMimeType(ContentService.MimeType.JSON);
}
पायरी ३: प्रोजेक्टला Deploy -> New Deployment करा. Gear आयकॉन वरून Web App निवडून 'Who has access' मध्ये "Anyone" करा. मिळालेली Web App URL वरील सेटिंग मध्ये पेस्ट करा.
